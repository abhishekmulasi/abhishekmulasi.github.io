" collection: portfolio ---

I built a video retrieval system that lets users search for videos based on their actual content instead of just titles or tags. The system works by extracting important visual features from videos and finding similar ones using those features. I also added machine learning models like SVM and k-NN to automatically classify and tag videos, making it easier to organize and search through large collections without relying on manual labeling.

To make the system fast and scalable, I implemented an approximate nearest neighbor search using Locality-Sensitive Hashing (LSH), which helps retrieve similar videos quickly even from large datasets. I also added a feedback loop where the system learns from user interactions and improves its search results over time. This project gave me hands-on experience with machine learning, handling high-dimensional data, and building systems that are both efficient and practical.
